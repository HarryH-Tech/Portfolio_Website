Technology Stocks and News

I built this website to allow me to view the latest business and technology news and also to give me an easy interface to be able to see the financial ratios of some technology companies. It uses Vue and axios to access the IEX API and iterate through an array of companies and then puts the information into a table, styled in Bootstrap. In the future I hope to find an API that has more options for viewing the financial results of companies so I can add more columns to the table. I also hope to add some sorting functionality to allow the user to sort the table results based on the value in each column.
